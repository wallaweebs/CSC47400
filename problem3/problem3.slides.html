Data Set Types of Different Graphs¶

1D Time Series Graph¶

The data shown in this graph represents the number of monthly sales of three types of products for a company over six months in 2021.

This graph is considered a 1D Time Series graph because each product's sales, being the only variable tracked with time, are recorded over equally spaced time intervals.

2D Scalar Field Graph¶

This graph shows spatial coordinates both axes (West-East and North-South positions) that represent the area around the Maunga Whau volcano in a 10m grid The colors represent the elevation of each coordinate, with purple being lower elevations and red being higher elevations.

The volcano map is a 2D scalar field graph because it takes two spatial coordinates, x and y, and returns one scalar value (elevation) at each point.

3D Scalar Field Graph¶

This graph shows a curved surface with varying scalar values represented by colors and a grid showing the curves of each area. There are 3 axes representing x, y, and z coordinates. The purple regions represent lower scalar values while orange regions represent higher scalar values.

This graph is a 3D scalar field since there are three independent spatial coordinates in the axes, the data exists in three-dimensional space, the colors represent scalar values at each point, and the mapping of the data reveals the full 3D grid structure it has.

2D Vector Field Graph¶

This graph shows the z-component of a magnetic field (Bz) in the x-y plane. The arrows represent the direction of the magnetic force while the color represents the strength of each area within this field.

This graph is a 2D vector field since there are 2 axes representing coordinates and a vector with both magnitude and direction shown by the arrows.

3D Shape¶

This graph shows a 3-dimensional sphere with a grid showing the shape of this.

This figure shown in this graph is a 3D shape because there are 3 axes, x, y, and z, for creating each point of the figure.

Graph¶

This is a bar graph that shows the number of children that listed a color as their favorite for different colors.

This is considered a graph because there are 2 axes, x and y, that represent different colors and the number of children, respectively. There is also a title and axis labels that provide context
